# nano-claude v0.6.0 — 记忆系统

**约 1,630 行 TypeScript，17 个文件。**

## 快速开始

```bash
npm install
cp .env.example .env
# 编辑 .env 填入你的 API key
npm run dev
```

## 本版本新增（相比 v5）

- **`src/memory.ts`** — 持久化键值记忆
- **多层级 CLAUDE.md** — 四级加载
- **`/remember`**、**`/forget`**、**`/memory`** 命令
